# MindChain.vn

Website của **MindChain** — đơn vị công nghệ & đào tạo, nghiêng về các dịch vụ
xoay quanh **công nghệ AI**. *Powered by Tinhvan Group.*

## Tổng quan
- Static HTML site (HTML5 / Bootstrap / GSAP / Swiper / AOS).
- Giao diện theo phong cách **dark "AI Agency"** (template Inotek home-8), đã
  Việt hóa toàn bộ và thay nội dung sang MindChain.
- Brand chính: tech blue `#1053f3`.

## Chạy thử cục bộ
```bash
# mở trực tiếp
start index.html
# hoặc serve tĩnh
python -m http.server 8099
# -> http://localhost:8099
```

## Cấu trúc
```
index.html              # Trang chủ (AI services)
```
